### Apa itu Bootstrap?
<p align="justify">
Bootstrap adalah framework front-end open-source yang paling populer di dunia untuk membangun website yang responsif dan mobile-first. Framework ini menyediakan kumpulan komponen CSS siap pakai, 
sehingga developer dapat membuat website dengan lebih cepat dan efisien tanpa harus menulis kode CSS dari nol. Bootstrap menggunakan sistem grid fleksibel dengan 12 kolom, yang memudahkan pembuatan 
layout responsif dan menyesuaikan tampilan secara otomatis di berbagai ukuran layar, mulai dari smartphone hingga desktop.
</p>

<p align="justify">
Bootstrap menyediakan berbagai komponen pre-designed seperti navigation bar, buttons, forms, cards, modals, dan masih banyak lagi, yang sudah di-styling dengan baik dan konsisten. 
Selain itu, Bootstrap dilengkapi dengan utility classes yang memudahkan pengaturan spacing, colors, typography, dan positioning tanpa perlu menulis CSS tambahan. Dengan dokumentasi 
yang lengkap dan komunitas yang besar, Bootstrap menjadi pilihan utama bagi developer pemula maupun profesional untuk membangun website modern yang responsif, konsisten, dan cross-browser compatible.
</p>
